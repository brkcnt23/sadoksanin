# Sadoksan ERP — Oturum Özeti (2026-06-05)

## ✅ Bu Oturumda Yapılanlar

### 1. SSH & GitHub Fix
- `id_ed25519` key'i GitHub deploy key olarak tanımlıydı (read-only), push yapamıyordu
- `github-key-1` push yetkisine sahip → `git config --local core.sshCommand "ssh -i ~/.ssh/github-key-1 -o IdentitiesOnly=yes"` ile repo'ya tanımlandı
- Remote ile local arası merge yapıldı (SSL dosyaları chown, docx/docx-content geçici silme)
- **12 commit pushlandı**, local-remote senkronize

### 2. API Schema Fix
- `apps/api/prisma/schema.prisma`'ya `isFeatured Boolean @default(false)` kolonu eklendi
- `@@index([isFeatured])` eklendi
- Build hatası giderildi (TypeScript `isFeatured does not exist in ProductWhereInput`)

### 3. Header "Ürünler" Menüsü
- **Hover tamamen kaldırıldı** — sadece tıkla açılır/kapanır
- `toggleMega()` basitleştirildi, `megaLocked` state'i kaldırıldı
- `onNavMouseLeave`, `onMegaTriggerEnter` temizlendi

### 4. Mega Menü — Kategori Görsel Paneli
- Her ana kategori ve sub kategori için **farklı görsel** (API'den 100 ürün çekilip benzersiz dağıtılıyor)
- Sub kategori hover'ında **teknik çizim tarzı ölçü çizgileri**:
  - Üst kenarda yatay çizgi + ok uçları → genişlik (örn: 60 cm)
  - Sağ kenarda dikey çizgi + ok uçları → yükseklik (örn: 120 cm)
- Ölçü çizgileri fade-in animasyonlu
- Görsel değiştirmek için `image` alanına URL yazmak yeterli

### 5. Sub Kategori Filtreleme
- Tüm sub kategoriler artık `subTo()` fonksiyonu ile parent kategori sayfasına `?ara=` filtresiyle yönleniyor
- Örn: "60 x 120 Seramikler" → `/kategori/seramik?ara=60%20x%20120`
- `/kategori/[slug].vue` → `ara` parametresini `/urunler` sayfasına aktarıyor
- `/urunler.vue` → `route.query.ara` ile arama kutusu otomatik dolduruluyor
- Her kategori için `slug` alanı eklendi

### 6. CMS Sayfaları UI Overhaul
- `SiteInfoPage.vue` komponenti oluşturuldu:
  - Hero header + breadcrumb
  - Temiz tipografi (prose), kart bazlı içerik
  - Mobil uyumlu, masaüstü sticky TOC
  - Alt kısımda WhatsApp + İletişim CTA
- `sayfa/[slug].vue` yeni komponent ile render ediliyor
- Yükleme ve hata durumları iyileştirildi

### 7. Proforma Kontrol
- API (dealer list, product search, proforma history) hepsi çalışıyor — Nginx üzerinden test edildi
- Admin panel rebuild edildi, 200 OK
