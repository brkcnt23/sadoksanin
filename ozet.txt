API ENDPOİNTLERİ (özet):

Auth:
  POST /auth/register     Kayıt (role: CUSTOMER|DEALER)
  POST /auth/login        Giriş → JWT
  GET  /auth/me           Profil (JWT)
  PATCH /auth/me          Profil güncelleme
  POST /auth/forgot-password  Şifre sıfırlama bağlantısı
  POST /auth/reset-password   Şifre değiştir

Products (public):
  GET  /products          Ürün listesi (+ discountedPrice + discount)
  GET  /products/:id      Ürün detay
  GET  /products/filters/categories
  GET  /products/filters/brands

Products (admin):
  GET  /products/admin/all
  POST /products          Oluştur
  PATCH /products/:id     Güncelle
  DELETE /products/:id    Sil
  POST /products/:id/visibility
  POST /products/:id/purchasable
  POST /products/:id/stock-thresholds
  GET  /products/admin/export   CSV
  POST /products/admin/import   CSV

Dealer (JWT):
  GET  /api/dealer/list
  GET  /api/dealer/profile
  GET  /api/dealer/cari/transactions
  GET  /api/dealer/cari/export       CSV
  GET  /api/dealer/reports/:type     CSV
  GET  /api/dealer/admin/list
  PATCH /api/dealer/:id/approve
  PATCH /api/dealer/:id/reject

CMS:
  GET  /cms/hero          Hero içeriği (public)
  PATCH /cms/hero         Hero güncelle (admin JWT)
  GET  /cms/settings      Ayarlar (admin JWT)
  PATCH /cms/settings     Ayarlar güncelle (admin JWT)

Discounts (admin JWT):
  GET    /discounts
  POST   /discounts
  PATCH  /discounts/:id
  DELETE /discounts/:id

Mailer (admin JWT):
  POST /mailer/send
  POST /mailer/cart-reminders?hours=24
